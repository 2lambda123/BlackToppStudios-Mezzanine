\subsection{Detailed Description}
\hyperlink{structThis}{This} is actor class for terrain. \hyperlink{structThis}{This} class is intended to be used for terrain. \hyperlink{structThis}{This} class uses sub-\/classes with fewer features, but still more then enough features for static terrain. \hyperlink{structThis}{This} helps reduce the memory footprint of the class as well as processing speed.
