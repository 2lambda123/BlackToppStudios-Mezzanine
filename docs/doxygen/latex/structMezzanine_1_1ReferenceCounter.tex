\subsubsection{Detailed Description}
\subsubsection*{template$<$class TypePointedTo$>$ struct Mezzanine::ReferenceCounter$<$ TypePointedTo $>$}

This exists once per object managed by a group of shared pointer to track items in memory. This exists to track the pointer to the managed object, and stores the single counter of existing references. Only one of these should be created for each group of pointers managing the same object. \begin{DoxyNote}{Note}
It should be extremely rare to need to create one of these outside of pointer implemenation. 
\end{DoxyNote}
