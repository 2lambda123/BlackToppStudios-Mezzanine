\subsubsection{Detailed Description}
This is a helper class for configuring physics settings of an \hyperlink{classMezzanine_1_1ActorRigid}{Actor\-Rigid}. 

This class contains functions for the configuring of physics specific settings of an \hyperlink{classMezzanine_1_1ActorRigid}{Actor\-Rigid}. This class can only configure the Actor\-Rigids physics. For configuring actor graphics, see Actor\-Graphics\-Settings.
