Detailed Description

This is the base class for all Primitive shapes.

A primitive shape is any shape that has a simple rule for determining it's bounds. As a rule all Primitive shapes are Convex. Primitive shapes are ideal for performace reasons and as such you should always try to appoximate objects with primitives, or a compound shape of primitives if at all possible.
