BlackTopp Studios Documentation

BlackTopp Studios inc. makes video games and video game components. Our flagship library game component is the Mezzanine Engine. Some components of the can be used on their own, like the DAGFrameScheduler a high performance multithreaded replacement for single threaded game loops.
