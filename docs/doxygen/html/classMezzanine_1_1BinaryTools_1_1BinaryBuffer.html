Detailed Description

A way to store and pass binary buffers like compiled bytcode.

Originally intended for use with ScriptCompilable as a basic way to store and pass bytecode around. This deletes the passed binary on destruction. to prevent this behavior set the Binary pointer to null.

This is designed to be fairly minimalistic and pointer-like so that it can be passed by value to reduce double pointer indirection and caching costs.
