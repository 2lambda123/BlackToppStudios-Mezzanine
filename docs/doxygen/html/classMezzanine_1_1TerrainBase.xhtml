Detailed Description

This is the base class from which all the terrains inherit.

The terrain classes store and manage all the relevant data regarding terrains inside the World. They serve as a binder between the physics and graphics for terrain and have functions that allow the manipulation of terrain loaded into the World. Currently there is 1 terrain class: MeshTerrain.
