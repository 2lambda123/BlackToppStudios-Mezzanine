Detailed Description

This is a widget used to determine spacing and ordering of other widgets(Cells) on a grid like surface.

This grid does support cells of different sizes, but all the cells of the same row/column will behave as if they share the size as the largest cell, in terms of spacing. The cells themselves will not be altered in size, and only the appropriate dimention will be applied(I.E. Each column will be spaced according to the largest width+spacing, and each row will be spaced according to the largest height+spacing). Thus it is ideal to have each cell be the same size, but not required.
