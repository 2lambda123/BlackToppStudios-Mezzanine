Detailed Description

template<typename T>
class Mezzanine::ReferenceCountTraits< T >

Every class that does not implement its own reference count gets the default one.

The Default reference count is not thread-safe, and requires that every dereferencing of the smart pointer is actually a double dereference. The double dereference is caused because the reference counter has to store a native pointer to the managed object.
