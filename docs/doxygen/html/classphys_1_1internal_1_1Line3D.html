Detailed Description

Does the bulk of the work that that the phys::LineGroup performs.

phys::LineGroup is a simple wrapper around this to perform precise low level interactions with Ogre, the rendering subsystem. This uses too much stuff from ogre to use publicly. so we need to hide it here in the phys::internal namespace.
