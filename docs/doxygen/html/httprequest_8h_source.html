54  /// @class HTTPRequest
55  /// @brief A simple class to facilitate the creation of HTTP requests.
56  /// @details Some of the data that can be set is mandatory, some of it is optional. The URI, major and minor versions, as well as
57  /// the request method to be used are mandatory in all cases. If you are using HTTP 1.1 or higher, then adding a "Host" header field
58  /// is also mandatory. The host should be the domain name of the server you want to request from. Some request methods make use of
59  /// the body, most don't. If the body is used then a header field may be required, such as sending a shoutbox message with a POST
60  /// request, which may require the message be put in the request body, and a "Content-Length" header be supplied with the size of
61  /// the body. Many more nuances exist, and we can't cover all use cases. So take care when using this class.
62  ///////////////////////////////////////
