185  /// @brief Sets the mode and scaler of auto-scaling applied to ListItems created by this ListBox.
186  /// @remarks There are a lot of considerations to be made with auto-scaling. Auto-scaling will adjust the size of the provided font
187  /// automatically based on the scaling mode that is set, which really decides what the relative scalar provided for auto-scaling is
188  /// relative to. If auto-scaling is relative to the screen, then the text will only change in size when the screen does. Parent and
189  /// layer relative scaling are very similiar since layers get their dimensions from the parent widget they belong to. The difference
190  /// comes from scaling applied to the layer itself. Parent relative scaling ignores the layer scaling, while layer relative factors
191  /// it in. @n @n
192  /// In addition to the relative component to the scaling, manual scaling will also be applied after the auto-scaling is. Auto-scaling
193  /// will be completely ignored if the custom size on a character is set.
194  /// @param Mode A ScalingMode enum value describing what the provided scaler will be relative to.
195  /// @param Scalar The relative value that will be used to determine the needed scaling to be applied to text in this layer.
196  virtual void SetListItemTextScale(const TextLayer::ScalingMode Mode, const Real Scalar);
