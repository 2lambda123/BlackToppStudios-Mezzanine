enum phys::GameWindow::WindowFlags

A listing of potential options for configuring a game window during construction.

Brief descriptions of the values are as follows:
WF_Fullscreen: This enables fullscreen on the window.
WF_Hidden: This hides the window so that it isn't visible.
WF_VsyncEnabled: This enables vsync for the window.
WF_FSAA_2: Enables Fullscreen Anti-Aliasing level 2 for the window.
WF_FSAA_4: Enables Fullscreen Anti-Aliasing level 4 for the window.
WF_FSAA_8: Enables Fullscreen Anti-Aliasing level 8 for the window.
WF_FSAA_16: Enables Fullscreen Anti-Aliasing level 16 for the window.
WF_Resizeable: Creates a window with resizable borders, otherwise it is fixed size.
WF_Minimized: Minimizes the window to the task bar immediately after construction.
WF_Maximized: Maximizes the window immediately after construction.
WF_Borderless: Removes all window decorations from the window(titlebar, borders, etc.).
