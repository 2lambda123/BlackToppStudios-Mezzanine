Detailed Description

This is a small extention of the WorldObject class that adds features common to non-static objects.

Many world objects are capable of being moved and oriented dynamically and at run time, but there are some objects that you don't want to have the ability or API to do so as it may break the intent of the object. This class is here to be avoided for those objects, and instead inherit directly from the WorldObject class while all other's inherit from this class. It's important to note that this class doesn't forbid static objects as the name may suggest, but rather it makes static status optional instead of forcing them to be static.
