Detailed Description

This namespace is for all the classes that facilitate the procedural creation of art assets.

Our Procedural Graphics system is based on the OgreProcedural library.

Enumeration Type Documentation

enum Mezzanine::Graphics::Procedural::BooleanOperation

An enum used to describe which boolean operation to take when processing two triangle buffers.

Enumerator
BO_Union 

Completely merges the the two buffers.

BO_Intersection 

Gets only the parts of the two buffers that are overlapping.

BO_Difference 

Gets only the parts of the two buffers that aren't overlapping.
